# The 'src' directory
The 'src' directory contains all the libraries' header files. Only the following header files are intended to be included by the user:
- I2CWrapper.h
- gyro.h
- gyros.h
- gyros/L3G4200D.h
- accelerometer.h
- accelerometers.h
- accelerometers/ADXL345.h
- quaternion.h
- euler.h
- vector3.h
